FitSwipe — Discover Your Aesthetic
An interactive web application to explore personal fashion style through swipe-based interactions
 Live Demo:
https://akshitahub.github.io/FitSwipe/
 Problem Statement
People struggle to identify their personal aesthetic while shopping or exploring fashion styles.
 Solution
FitSwipe allows users to swipe through fashion items, like what resonates with them, and:
Build a personalized closet
Discover their dominant aesthetic
Get a celebrity style match
All in a fun, pressure-free interface.

 Key Features

Swipe-based fashion exploration
Personalized greeting & profile
Closet of liked items
Celebrity style match
Aesthetic vibe summary
Responsive, single-page UI

 Tech Stack

HTML – Structure
CSS – Styling & responsiveness
JavaScript – Logic & interactivity
GitHub Pages – Deployment

 How It Works

User enters name and logs in
Items are shown one by one
Likes & skips update preferences
Liked items appear in Closet
Dominant vibe & celeb match are displayed
Tabs switch sections without page reload

 Future Scope

Swipe gestures & animations

Shareable aesthetic profiles
